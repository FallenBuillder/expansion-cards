**DVI (Digital  Interface)**

DVI-A
• Sygnał: Analogowy
• Transmisja: Równoległa
• Taktowanie: Synchroniczne
DVI-D (Single Link)
• Sygnał: Cyfrowy
• Transmisja: Szeregowa
• Taktowanie: Synchroniczne
DVI-D (Dual Link)
• Sygnał: Cyfrowy
• Transmisja: Szeregowa
• Taktowanie: Synchroniczne
DVI-I (Single Link)
• Sygnał: Cyfrowy i Analogowy
• Transmisja: Szeregowa i Równoległa
• Taktowanie: Synchroniczne
DVI-I (Dual Link)
• Sygnał: Cyfrowy i Analogowy
• Transmisja: Szeregowa i Równoległa
• Taktowanie: Synchroniczne


all DVI types are
Type: Output
